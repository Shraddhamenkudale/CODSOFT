# To-Do List Application

This project is a simple command-line based To-Do List application 
created using Python as part of my CodSoft Internship (Task 1).

The main purpose of this project is to manage daily tasks in an easy 
and organized way using basic Python concepts.

## Features

- Add new tasks
- View all tasks
- Update existing tasks
- Mark tasks as completed
- Delete tasks
- Simple and user-friendly menu system

## Technologies Used

- Python
- Command Prompt (Windows)

## How to Run the Program

1. Make sure Python is installed on your system.
2. Open Command Prompt.
3. Navigate to the project folder.
4. Run the following command:

   python todo.py

5. Follow the on-screen instructions.

## About This Project

This project was developed to practice basic Python concepts such as:
- Lists
- Functions
- Loops
- Conditional statements
- User input handling

It is a simple console-based application created for learning 
and internship purposes.
