# Rock Paper Scissors Game

This project is a simple command-line based Rock Paper Scissors game 
created using Python as part of my CodSoft Internship Task.

The purpose of this project is to implement basic game logic 
where a user plays against the computer.

## Features

- User selects rock, paper, or scissors
- Computer generates a random choice
- Displays result (Win / Lose / Tie)
- Score tracking for multiple rounds
- Option to play again
- Simple and interactive menu

## Technologies Used

- Python
- Random module
- Command Prompt (Windows)

## How to Run the Program

1. Make sure Python is installed on your system.
2. Open Command Prompt.
3. Navigate to the project folder.
4. Run the following command:

   python rock_paper_scissors.py

5. Enter your choice (rock, paper, or scissors) and play the game.

## About This Project

This project was developed to practice basic Python concepts such as:
- Loops
- Conditional statements
- User input handling
- Random number generation
- Score tracking using variables

It is a simple console-based game created for learning 
and internship purposes.
