# Contact Book Application

This project is a simple command-line based Contact Book application 
created using Python as part of my CodSoft Internship Task.

The purpose of this project is to manage contact details such as 
name, phone number, and email using basic Python concepts.

## Features

- Add new contact
- View all contacts
- Search contact by name
- Update existing contact
- Delete contact
- Simple menu-driven system

## Technologies Used

- Python
- Command Prompt (Windows)

## How to Run the Program

1. Make sure Python is installed on your system.
2. Open Command Prompt.
3. Navigate to the project folder.
4. Run the following command:

   python contact_book.py

5. Choose an option from the menu and follow the instructions.

## About This Project

This project was developed to practice basic Python concepts such as:
- Lists
- Dictionaries
- Functions
- Loops
- Conditional statements
- User input handling

It is a simple console-based application created for learning 
and internship purposes.
