# Simple Calculator Application

This project is a simple command-line based Calculator 
created using Python as part of my CodSoft Internship (Task 2).

The main purpose of this project is to perform basic arithmetic 
operations using user input and simple Python logic.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles invalid input
- Prevents division by zero error
- Simple menu-driven system

## Technologies Used

- Python
- Command Prompt (Windows)

## How to Run the Program

1. Make sure Python is installed on your system.
2. Open Command Prompt.
3. Navigate to the project folder.
4. Run the following command:

   python calculator.py

5. Choose an operation from the menu and enter two numbers.

## About This Project

This project was developed to practice basic Python concepts such as:
- Functions
- Conditional statements (if-else)
- Loops
- User input handling
- Error handling using try-except

It is a simple console-based calculator created for learning 
and internship purposes.
